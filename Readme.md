# Diabetes Risk Prediction using Machine Learning

This project develops machine learning models to predict **Type 2 Diabetes Mellitus (T2DM)** risk using data from the **National Health and Nutrition Examination Survey (NHANES) 2007–2018**.

The objective of this study is to evaluate **discrimination, calibration, and temporal stability** of predictive models for diabetes risk prediction.

---

## Models Used

The following models were implemented and evaluated:

- Logistic Regression
- Gradient Boosting (XGBoost)
- Probability-Level Stacked Ensemble

---

## Evaluation Metrics

Model performance was evaluated using:

- AUROC (Area Under ROC Curve)
- Brier Score
- Calibration Slope
- Temporal Validation
- Subgroup Analysis

---

## Dataset

The dataset used in this project is derived from the **NHANES survey (2007–2018)**.

Key variables include:

- Age
- Sex
- Race/Ethnicity
- Body Mass Index (BMI)
- Blood Pressure
- Glycohemoglobin (HbA1c)
- Poverty Income Ratio
